Este repositório contém todo o código, base teórica e o artigo final do nosso trabalho de regressão aplicada ao conjunto de solubilidade apresentado no curso de Inteligência Computacional Aplicada (ICA).
O objetivo geral foi explorar diferentes modelos de regressão, desde o OLS simples até métodos penalizados para prever a solubilidade de compostos químicos a partir de descritores moleculares.
Usamos o datase do livro Applied Predictive Modeling (Kuhn & Johnson), que contém 1.267 compostos, cada um descrito por:

- 208 fingerprints binários (presença/ausência de subestruturas químicas)

- 16 contagens químicas (ex.: número de ligações, cloros, bromos etc.)

- 4 descritores contínuos (massa molecular, área de superfície...)

- 1 alvo: valor experimental de solubilidade 

Códigos usados estão na pasta src, base teóricas e explicações aprofundadas se encontão na pasta notebooks.
